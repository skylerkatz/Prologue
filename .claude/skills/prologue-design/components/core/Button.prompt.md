Prologue button; use `primary` for the one main action on screen (orange), `secondary` for toolbar buttons (Refresh, Export), `outline` for "+ Add comment", `ghost` for comment-header text actions (Reply/Resolve/Edit), `ghost-danger` for destructive confirms.

```jsx
<Button variant="primary">+ Add review comment</Button>
<Button variant="secondary" size="sm">↻ Refresh</Button>
<Button variant="ghost" size="sm">Resolve</Button>
```
