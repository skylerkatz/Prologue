2. **Tokens.** Replace the `:root` custom-property values in `src/App.css` with the values from `tokens/colors.css` — the names already align (`--bg`→`--bg-app` semantics, `--border`, `--accent`, `--diff-add-bg`, …). Add the new tokens the mockup needs (`--surface-selected`, `--line-selected-bg`, badge tint pairs, `--teal-800`, `--text-on-dark`, `--shadow-menu`). Delete the `prefers-color-scheme: dark` block — Prologue is light-only for now.
3. **Title bar.** Teal custom title bar (Tauri: `titleBarStyle` overlay or `decorations: false` + drag region): traffic lights, orange ribbon mark (extract the ribbon path from `assets/prologue-logo.svg`), Lora "Prologue" wordmark, "— {branch}" muted. Update `tauri.conf.json` window title and the app icon set from the logo SVG.
4. **Toolbar.** Repo button → teal pill with cream mono text. Base/Branch selects → mono, `--border-strong` outline. ModeToggle → segmented on `--bg-subtle`, selected segment white with 7px orange dot + 600 weight (see `components/core/ModeToggle.jsx`). 6. **Diff pane.** Each file becomes a bordered white card (8px radius, `--shadow-card`) with the header from `components/diff/FileCardHeader.jsx`; "+ Add comment" → outline variant. 7. **Comments.** Restyle `CommentCard` per `components/comments/`: 3px orange left rail (roots only), teal circle avatar with mono `C{id}`, bold "You", muted meta, ghost actions right-aligned, body 13.5px sans. 8. **Welcome page.** Centered logo (72px, 16px radius), Lora "Prologue" 28px, tagline, orange primary "Open Repository…", recents card list with mono name/path (see the kit's Welcome screen).
9. **Archive overlay + export menu + toast.** Scrim `--overlay-scrim`; panel 10px radius + `--shadow-menu`; "Archived reviews" heading in Lora; menu items hover `--bg-subtle`; toast per `components/core/Toast.jsx`.
10. **Hide-whitespace toggle** (new feature). Toolbar button right of ModeToggle, styled like a selected segment when active (orange dot, 600 weight, white bg). Global across all files. Backend: pass `ignore_whitespace` into the Rust diff (git2 `DiffOptions::ignore_whitespace`), or filter whitespace-only line pairs in the summary/file-diff path; render an italic "N whitespace-only lines hidden" band where lines were suppressed. Persist app-wide (tauri-plugin-store or the existing recents mechanism) so it survives restarts and applies to every review.

Constraints: don't change behavior except step 10; keep all existing class names where practical (tests/automation may reference them); exact values come from the skill's CSS files — never approximate; verify each screen against `ui_kits/prologue/index.html` side by side before moving on.
